Overview

We estimate the probability of rolling a specific Echo in Wuthering Waves based on its selected substats and substat values. It does not account for main stat rolls, secondary stats (which are fixed by cost), or the Echo set/type itself.

What each Echo card shows

Every Echo card displays the following information:

Field Description
Cost 4, 3, or 1. Determines which main stats and secondary stats are available.
Main Stat Selected by the user. Not included in the probability calculation.
Secondary Stat Fixed by cost (ATK 150 for 4-cost, ATK 100 for 3-cost, HP 2280 for 1-cost). Not included in the probability calculation.
Substats (up to 5) Each substat has an attribute and a rolled value. Both are included in the probability calculation.
Substats Roll Probability The chance of rolling the selected substat attributes, regardless of their values.
Roll Probability With Values The chance of rolling the selected substat attributes and their specific values.

Substat attribute probability

Each time a new substat is rolled, one attribute is chosen from the remaining available attributes. Because duplicate substats are not allowed, the chance of rolling a specific remaining attribute increases as more substats are already present on the Echo.

Current substats on Echo Chance of rolling one specific attribute
0 substats 7.6923%
1 substat 8.3333%
2 substats 9.0909%
3 substats 10.0000%
4 substats 11.1111%

There are 13 possible substat attributes. We treat the final set of substats as unordered — the order in which they were rolled does not matter.

Substat value probability

After a substat attribute is selected, the game rolls a value for that attribute. Each attribute has its own value table with different probabilities for each possible value.

For example, CR can roll 6.30%, 6.90%, 7.50%, 8.10%, etc., each with its own probability. We multiply the attribute roll probability by the probability of the selected value.

When the roll probability with values is extremely small (below 0.000001%), the "Extremely low" message is displayed instead of a scientific-notation number.

Formulas

Substats Roll Probability

For an Echo with k selected substats:

P(attributes) = k! × Πi=0..k-1 select_prob[i]

Where select_prob[i] is the probability of rolling a specific attribute when i substats already exist on the Echo. The k! factor accounts for all possible roll orders that produce the same final set.

Roll Probability With Values

P(attributes + values) = P(attributes) × Πj=1..k value_prob[j]

Where value_prob[j] is the probability of rolling the selected value for the j-th substat attribute.

Why there is no combined probability

Even a single Echo with several specific substats and values can have a very low probability. Multiplying multiple Echo probabilities together produces numbers that are technically correct but not useful for most users. Because of that, each Echo card shows its own roll probabilities separately.
